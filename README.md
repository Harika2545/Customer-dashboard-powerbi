# 📊 Customer Dashboard - Power BI

## 📌 Project Overview
This project is an interactive Customer Dashboard developed using Power BI to analyze customer demographics, purchasing behavior, sales performance, and key business metrics. The dashboard enables data-driven decision-making through clear visualizations and interactive reports.

---

## 🎯 Objectives
- Analyze customer demographics and behavior
- Track sales and revenue performance
- Identify high-value customers
- Monitor key performance indicators (KPIs)
- Support business decisions with interactive visualizations

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV

---

## 📈 Dashboard Features
- Executive Summary
- Customer Analysis
- Sales Performance
- KPI Cards
- Interactive Slicers
- Dynamic DAX Measures
- Drill-through Analysis

---

## 💡 Key Insights
